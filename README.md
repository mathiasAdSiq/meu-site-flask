# meu-site-flask
Site de Gerencialmente de Estoque de Bebidas e ou alimentos


# Controle de Estoque

Sistema simples de controle de estoque feito com Python e Flask.

## Funcionalidades

- Cadastro de produtos
- Entrada e saída de itens
- Estoque mínimo
- Previsão de compra baseada no consumo dos últimos 7 dias

## Como rodar

```bash
python -m pip install -r requirements.txt
python app.py
